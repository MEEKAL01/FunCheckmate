 ♟️ Online Chess Web App

This is a simple online chess web application built as a beginner project. It allows users to play chess in the browser in a fun and interactive way.

📌 About the Project

I created this project to practice my web development skills and understand how game logic works. It’s a basic version, and I plan to improve it as I gain more experience.

 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎮 Features

* Play chess in the browser
* Basic game functionality
* Simple and clean interface

 🚀 Future Improvements

* Add multiplayer functionality
* Improve game logic and rules
* Better UI/UX design

 📥 How to Run

1. Download or clone the repository:
2. Open the project folder
3. Open `index.html` in your browser
